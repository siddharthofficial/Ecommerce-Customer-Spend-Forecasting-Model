##  Project Overview

**Goal:**  
Predict how much a customer will spend yearly based on features like session duration, app/website usage, and membership length.

**Dataset:**  
500 customer records with features like:
- Session Length
- Time on App
- Time on Website
- Membership Duration
- Yearly Amount Spent (target)
